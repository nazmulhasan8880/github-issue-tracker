**var, let, const:**
var > is function scoped, let > is block scoped and const > cannot be reassigned.

**Spread operator (...):**
It is used to expand or copy array or object elements.

**map(), filter(), forEach():**
map() - returns a new array, filter() - returns matched items and forEach() - just loops.

**Arrow function:**
A short way to write a function using =>.

 **Template literals:**
Strings written with backticks ( ) that allow ${variable} inside.
